# モザイク画 CSV/Excel変換プログラム
**北見北斗高校 生徒会執行部&emsp;文化祭 【全校制作】**

このプログラムは「モザイク画データ CSV/Excel変換プログラム」です。
**bigart.exe** から出力されたデータを各クラスに配布しやすい形式のCSV/Excelに変換します。
